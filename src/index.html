Git Prácticas de uso básico

    # 1. Inicializar repositorio
    $ git init

    # 2. Monitorear cambios en el repositorio
    $ git status

    # 3. En caso de haber cambios, agregarlos para guardar versión
    $ git add -A

    # 4. Guardar la versión
    $ git commit -m "Primera versión"

    # Los pasos 2, 3 y 4 se repiten cuantas veces sea necesario, o bien,
    # cuantas veces existan archivos nuevos o cambios en los actuales.
